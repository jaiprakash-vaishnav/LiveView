# 🎥 LiveView

Full stack Video call Platform

## Description

Developed a real-time video calling web application that allows users to create and join meetings instantly, featuring live video streaming, chat functionality, and responsive UI for a seamless communication experience.

🔗 [Live Demo]()

## 🚀 Getting Started

Follow these steps to run LiveView locally:
## Node.js is also required.
### 1️⃣ Clone the Repository

```bash
git clone 
cd LiveView
```
For Backend
```
cd backend
```
For frontend
```
cd frontend
```

### 2️⃣ Install Dependencie
```
npm install
```

### 3️⃣ Set Up Environment Variables
Create a .env file in the root directory of backend and add the following 
```
PORT=8000
MONGODB_ATLAS=your_cloud_host_mongoDb_url
```

### 4️⃣ Start MongoDB
Ensure MongoDB is running locally or provide a remote connection string in your app.
### 5️⃣ Run the Application
for backend
```
node src/app.js
```
for frontend 
#### guide make in frontend environment.js is_Prod is false
```
npm run dev
```

🔗 [Visit in your browser.](http://localhost:5173).
